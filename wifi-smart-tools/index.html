Overview

WiFi Smart Tools is a comprehensive network utility application designed to help you analyze, manage, and optimize your WiFi connections. This Privacy Policy comprehensively discloses how our app accesses, collects, uses, and shares user data.

Quick Summary:

  • No Personal Data Collection: We do not collect or store personal information such as names, emails, or phone numbers on our own servers

1. Information We Collect

1.1 Network Information

When you grant location permission, we collect:

  • WiFi network SSID (network name)
  • IP address (local network)
  • Signal strength (RSSI)
  • Gateway IP address (router)
  • Subnet mask information
  • WiFi frequency band (2.4GHz/5GHz)
  • Channel information
  • Security type (WPA2/WPA3)
  • Connected devices on your network (IP addresses, MAC addresses, device types, manufacturers)

⚠️ Important: This data is collected ONLY when you actively use the app and is NOT stored or transmitted to any server. Device scanning uses actual network ping (ICMP) to detect reachable devices and retrieves MAC addresses from ARP table for device identification. All scanning happens locally on your device.

1.2 Speed Test Results

When you run a speed test:

  • Download speed (Mbps)
  • Upload speed (Mbps)
  • Ping/latency (ms)
  • Test timestamp

✓ Stored: Locally on your device only
✓ Never uploaded to any server

1.3 App Preferences

We store your app settings locally:

  • Dark/Light theme preference
  • Onboarding completion status
  • App settings and preferences

2. What We DO NOT Collect

We NEVER collect:

  • ❌WiFi passwords or credentials
  • ❌Your physical location or GPS coordinates
  • ❌Personal identification information (name, email, phone)
  • ❌Browsing history or internet activity
  • ❌Files or photos from your device
  • ❌Contacts or call logs

3. How We Use Your Information

•

Network Analysis: Display WiFi networks, signal strength, and channel information

•

Device Scanner: Scan and identify devices connected to your network using ICMP ping and ARP table lookup. Collects local IP addresses, MAC addresses (from device's ARP cache), and attempts to identify device types and manufacturers. All scanning is performed locally; no data is transmitted to external servers.

•

Speed Testing: Measure and display your internet connection speed

•

App Functionality: Provide features like signal meter, QR code generation (for WiFi sharing), and network diagnostics

•

Improve App Performance: Maintain your preferences for better user experience

🔒 All data processing happens locally on your device. We do NOT upload any network data, device information, or scan results to external servers.

4. Location Permission Explained

Why We Need Location Permission

Android OS requires location permission to access WiFi network names (SSIDs). This is a system requirement, not our choice.

✓ What we do with it:

  • Read WiFi network names (SSIDs)
  • Detect signal strength
  • Analyze network channels

✗ What we DON'T do:

  • Track your physical location or GPS coordinates
  • Store or transmit location data
  • Monitor your movements

7. Data Storage & Security

Data Storage:

All data collected by WiFi Smart Tools is stored locally on your device only:

  • Speed test history (SQLite database on your device)
  • App preferences (Android SharedPreferences)
  • No cloud backups or server-side storage

Security Measures:

  • Local Processing: All network analysis happens on your device
  • Minimal Internet Usage: Many core WiFi analysis features work locally on your device. Internet is required only for features such as Speed Test and advertisements
  • Encrypted Storage: Android's built-in encryption protects your data
  • No Permissions Abuse: We only request necessary permissions for network analysis
  • Network Scanning: Device scanning uses standard Windows ping command (ICMP) and ARP table lookup for device detection
  • MAC Address Privacy: MAC addresses are read from your device's local ARP cache and never transmitted
  • Open Source Libraries: We use trusted, open-source networking libraries

Data Deletion:

You can delete all app data at any time:

  • Clear History: Use "Clear History" button in app settings
  • Uninstall App: Uninstalling removes all data permanently
  • Device Settings: Settings → Apps → WiFi Smart Tools → Storage → Clear Data
